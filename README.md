

My very first web development project, built using only HTML and CSS while learning the fundamentals of web design and page layouts.

This project is a multi-section gaming-themed landing page featuring a hero section, services, pricing plans, team members, testimonials, and contact information.

Although simple, this project helped me understand how websites are structured and how to create responsive layouts using modern CSS techniques.

## 🚀 Live Demo

https://sudip-project-01.netlify.app/

## ✨ Features

- Responsive landing page layout
- Hero section with call-to-action
- Services and feature showcase
- Pricing plans section
- Team members section
- Customer testimonials
- Contact information section
- Clean and organized design

## 🛠️ Built With

- HTML5
- CSS3

## 📚 What I Learned

Through this project, I learned:

- HTML page structure and semantic elements
- CSS styling and layout techniques
- Working with images and icons
- Creating reusable page sections
- Responsive design basics
- Deploying websites online

## 🎯 Project Purpose

The goal of this project was to practice front-end web development fundamentals and gain hands-on experience building a complete landing page from scratch.
